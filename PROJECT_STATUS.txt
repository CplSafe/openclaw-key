```bash

```

## API 端点

- GET  /api/install/:token.sh    - 获取验证脚本（Unix）
- GET  /api/install/:token.ps1   - 获取验证脚本（Windows）
- POST /api/verify-install       - 验证并返回真实安装脚本
- Query getLicenses              - 获取所有卡密
- Action createLicense           - 创建卡密
- Action revokeLicense           - 撤销卡密
